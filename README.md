# Travel Site

A responsive travel website built with Next.js.  
The project was created as a learning exercise to practice modern frontend development, routing, component structure, and responsive design.

---

## Live Demo

https://travel-site-kappa-drab.vercel.app/

---

## Tech Stack

- Next.js
- React
- CSS Modules
- JavaScript (ES6+)
- Next.js Image Optimization
- Vercel (deployment)

---

## Features

- Responsive layout (desktop, tablet, mobile)
- Dynamic routing (blog pages)
- Interactive UI components
- Video preview section with lazy loading
- Image optimization using Next.js Image component
- Modular component structure
- Dark/light header logic based on route

---

## Project Structure

- components/ – reusable UI components
- app/ – Next.js routing (pages)
- public/ – static assets (images, icons)
- styles/ – CSS modules

---

## Deployment

The project is deployed using Vercel with automatic CI/CD connected to GitHub.
Each push to the main branch triggers a new deployment.

---

## Notes

This is a learning project and is still under development.  
Some features (such as multilingual support) were intentionally removed to simplify the final version.
